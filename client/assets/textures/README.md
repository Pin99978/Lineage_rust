# Visual Assets (Phase1 Task03)

Place these files here:

- `player.png`: 2D sprite sheet for character animation.
  - Expected grid (MVP default): 3 columns x 32 rows, frame size 64x64.
  - Row groups: Idle(0..7), Walk(8..15), Attack(16..23), Dead(24..31).
- `map_bg.png`: world background texture.

If missing, the client falls back to colored primitives and still runs.
